jQuery

Selectors

CSS uses selectors to reference HTML tags (i.e. p.className). jQuery has copied CSS and uses the same CSS selectors to reference HTML tags.

Goal

Keep the appearance the same for the user. We will convert the CSS to jQuery. Both CSS selectors and CSS rules work in jQuery with different syntax.
