Slippy Map

Basic Map

Goal

    • As a user, I want to see my business on the map.
    • As a dev, using a map API add a marker or pin on the slippy map.
    • As a user, I want to see my business on the map and display our logo when pin is clicked.
    • As a dev, using a map API add a marker or pin on the slippy map. The pin's infobubble will contain the image HTML element.

Technologies

  • Google Maps JS API v3
  • JavaScript
  • CSS
  • HTML
