Slippy Map

Map with Flickr pins

Goal

Display a slippy map with a many pins representing photos found on Flickr. When user clicks on a pin, the photo will display in the info bubble.

Technologies

  • Flickr API
  • Google Maps JS API v3
  • JavaScript
  • jQuery AJAX
  • CSS
  • HTML

Resources

  • Flickr API documentation for geo photos
  • Google Maps JavaScript API v3
